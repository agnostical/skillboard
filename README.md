# Skillboard

A collection of portable AI skills, usable with any client or model. Soon, it will also include a curated set of references to other projects that gather and share skills.

## What's inside

- **Skills**: skills written for day to day use, each in its own folder under [`skills/`](skills/). For example, [`plan-review`](skills/plan-review/) reviews a plan or spec document for clarity, scope, sequencing, and consistency before it's executed.
- **References** (soon): links to other repositories and projects that collect, document, or distribute skills, so they're easy to find in one place.

## Structure

Each skill lives in its own directory containing a `SKILL.md` (the skill definition) and a license. Drop a skill folder into your AI client's skills directory to use it.
